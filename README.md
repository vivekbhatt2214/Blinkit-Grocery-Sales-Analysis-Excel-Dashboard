# 🛒 Blinkit Grocery Sales Analysis Dashboard

## 📌 Project Overview

This project presents a comprehensive sales analysis of Blinkit Grocery Store data using Microsoft Excel. The dashboard provides key business insights related to sales performance, customer ratings, outlet performance, product categories, and outlet distribution.

The objective of this project is to transform raw sales data into meaningful business insights through interactive visualizations and KPI tracking.

---

## 🎯 Business Objectives

- Analyze overall sales performance.
- Identify top-performing outlet types.
- Compare sales across different outlet sizes.
- Evaluate sales contribution by outlet location.
- Analyze customer preferences by item category.
- Monitor average sales and customer ratings.
- Generate actionable business insights for decision-making.

## 🔍 Key Insights

### 1. Sales Performance
- Total sales exceeded **$1.20 Million**.
- Average sales per transaction reached **$141**.

### 2. Outlet Size Analysis
- Medium-sized outlets generated the highest revenue contribution.
- High-sized outlets contributed significantly to total sales.

### 3. Location Analysis
- Tier 3 locations produced the highest sales.
- Tier 1 and Tier 2 locations showed steady performance.

### 4. Product Analysis
- Fruits & Vegetables emerged as one of the highest-selling categories.
- Snack Foods showed strong customer demand.
- Household products maintained consistent sales performance.

### 5. Outlet Type Analysis
- Supermarket Type 1 generated the highest sales.
- Grocery Stores contributed a significant share of revenue.
